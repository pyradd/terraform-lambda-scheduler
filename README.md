# terraform-iac

A minimal example of deploying a lambda funtion and an eventbridge scheduler with terraform IaC.

## Requirements

`terraform ">= 0.14.9"`

`aws cli`


## Usage
1. Intialize terraform

    `terraform init`

2. To plan deployment

    `terraform plan`

3. To deploy resources

    `terraform apply`

4. To destroy resources

    `terraform destroy`
